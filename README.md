# THE LAST NODE (Godot)

A pixel-art side-scrolling beat-em-up set in a dystopian CBDC future. Fight your way through the surveillance state to protect the last running Bitcoin node.

**Built with Godot 4.3. Bitcoin only. Never crypto.**

> "Running Bitcoin." — Hal Finney, January 10 2009

## Development

Requires [Godot 4.3+](https://godotengine.org/).

1. Clone the repo
2. Open `project.godot` in Godot
3. Hit F5 to play

## Controls

| Action | Key |
|--------|-----|
| Move   | WASD / Arrows |
| Attack | Z |
| Special | X |
| Jump   | C |
| Grab   | V |
| Pause  | Esc |
